## Button Labels

Use clear, action-oriented labels:

| Context | Label |
|---------|-------|
| Primary CTA | "Begin Your Ascent" |
| Save action | "Save" |
| Cancel action | "Cancel" |
| Add item | "Add Habit" |
| Continue flow | "Continue" |
| Complete flow | "Done" |
| Destructive | "Delete" (never "Remove" for permanent actions) |

## Questions We Ask

Summit uses questions to prompt reflection:

- "What does feeling your best look like to you?"
- "How did today's climb feel?"
- "What's one small step you can take tomorrow?"
- "What went well this week?"
- "What would you do differently?"
- "What are you grateful for today?"

---

## Recap: Voice Checklist

Before publishing any copy, ask:

- [ ] Is it encouraging, not demanding?
- [ ] Is it clear and concise?
- [ ] Does it avoid guilt or pressure?
- [ ] Does it sound human, not robotic?
- [ ] Does it use Summit vocabulary where appropriate?
- [ ] Is the tone right for the context?
