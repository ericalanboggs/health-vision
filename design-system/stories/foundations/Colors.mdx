## Usage Guidelines

### Do
- Use **Peak Lime** for primary CTAs — high-energy, impossible to miss
- Use **Vibrant Emerald** for active states, checkmarks, and success indicators
- Use **Rich Forest** for all primary text instead of pure black
- Use **Summit Mint** as the default page background
- Pair **Summit Sage** with **Rich Forest** text for tags and badges

### Don't
- Mix warm beige/stone colors with the Summit palette
- Use pure black (#000000) for text
- Overuse Peak Lime — reserve for the most important actions
- Use low-contrast color combinations

## Accessibility

All color combinations meet WCAG AA standards:

| Combination | Contrast Ratio | Status |
|-------------|----------------|--------|
| Rich Forest on White | 14.8:1 | ✅ AAA |
| Rich Forest on Mint | 13.2:1 | ✅ AAA |
| Rich Forest on Sage | 9.6:1 | ✅ AAA |
| Forest on Peak Lime | 8.2:1 | ✅ AAA |
| White on Emerald | 4.5:1 | ✅ AA |
| White on Moss | 5.2:1 | ✅ AA |
| White on Pine | 7.3:1 | ✅ AAA |
| Mint on Pine | 10.2:1 | ✅ AAA |
| White on Rich Forest | 14.8:1 | ✅ AAA |
