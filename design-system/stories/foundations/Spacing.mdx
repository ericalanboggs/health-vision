# Spacing & Layout

Summit embraces **generous whitespace** for a fluid, open canvas feel. We're moving from confined cards to expansive, breathing layouts.

---

## Touch Targets

## Z-Index Scale

| Token | Value | Usage |
|-------|-------|-------|
| `z.base` | 0 | Default |
| `z.raised` | 10 | Slightly raised |
| `z.dropdown` | 20 | Dropdown menus |
| `z.sticky` | 30 | Sticky headers |
| `z.fixed` | 40 | Fixed elements |
| `z.modalBackdrop` | 50 | Modal backdrop |
| `z.modal` | 60 | Modals |
| `z.popover` | 70 | Popovers |
| `z.tooltip` | 80 | Tooltips |
| `z.max` | 100 | Maximum (toasts) |

---

## Breakpoints

| Token | Value | Usage |
|-------|-------|-------|
| `sm` | 640px | Small tablets |
| `md` | 768px | Tablets |
| `lg` | 1024px | Small desktops |
| `xl` | 1280px | Large desktops |
| `2xl` | 1536px | Extra large screens |

---

## Container Widths

| Token | Value | Usage |
|-------|-------|-------|
| `narrow` | 640px | Narrow content (forms) |
| `default` | 768px | Standard content |
| `wide` | 1024px | Wide content |
| `max` | 1280px | Maximum width |

## Tailwind Classes

```jsx
// Spacing
className="p-4"      // 16px padding
className="m-6"      // 24px margin
className="gap-3"    // 12px gap

// Border radius
className="rounded-sm"    // 6px
className="rounded-lg"    // 12px
className="rounded-full"  // 9999px

// Shadows
className="shadow-card"
className="shadow-elevated"
className="shadow-modal"
className="shadow-focus"

// Z-index
className="z-10"     // raised
className="z-50"     // modal backdrop
className="z-60"     // modal
```
