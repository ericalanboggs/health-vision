## Installation

```bash
# In the design-system directory
npm install

# Start Storybook
npm run dev
```

### Using in Your App

```jsx
// Import the Tailwind preset
// tailwind.config.js
import summitPreset from '@summit/design-system/tailwind-preset'

export default {
  presets: [summitPreset],
  // ...
}

// Import components
import { Button, Card, Checkbox } from '@summit/design-system'

function MyComponent() {
  return (
    <Card>
      <Checkbox label="Morning meditation" />
      <Button>Complete Habit</Button>
    </Card>
  )
}
```

## Philosophy

> "We don't push you to the summit ‚Äî we walk alongside you on the ascent."

Summit believes in:

1. **Positive regard** ‚Äî Assume good intent, celebrate progress
2. **Wholeness** ‚Äî People have the answers within themselves
3. **Sustainability** ‚Äî Long-term habits over quick fixes
4. **Clarity** ‚Äî Clean interfaces, clear communication
5. **Uplifting** ‚Äî Encouraging language, vibrant visuals
